## 🔐 FILE PERMISSIONS (For Linux/Unix Hosting)

Set these permissions after uploading:

```bash
# HTML, CSS, JS, Image, Config Files: 644
chmod 644 index.html
chmod 644 css/styles.css
chmod 644 js/*.js
chmod 644 *.php
chmod 644 robots.txt
chmod 644 sitemap.xml

# Directories: 755
chmod -R 755 css/
chmod -R 755 js/
chmod -R 755 pages/
chmod -R 755 images/

# Make .htaccess readable
chmod 644 .htaccess
```
